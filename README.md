# Veblen

Editorial shopping website prototype built from the supplied mockup. The site recreates the paper-like catalog experience in React and TypeScript, with browse, product detail, methodology, and saved/alerts views.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Highlights

- Responsive editorial layout inspired by the attached Veblen mockup
- Interactive product catalog with filtering and search
- Product detail page with grading notes and price history
- Saved items, alert composer, and watch-history views
